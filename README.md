# OpenMe

OpenMe is an actionable bio page where people can create personal endpoints such as `/collaborate`, `/ask-me`, `/feedback`, and `/challenge-me`.

## MVP

- Public profile page
- Custom interaction endpoints
- Visitor submissions
- Owner inbox
- AI summaries and boundary checks
- Public machine-readable profile API

## Local development

```bash
pnpm install
docker compose up -d
pnpm dev
