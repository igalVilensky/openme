# OpenMe MVP Scope

OpenMe is an actionable bio page.

Users can create a public profile with personal interaction endpoints such as:

- GET /about
- GET /now
- GET /links
- POST /collaborate
- POST /ask-me
- POST /feedback
- POST /challenge-me
- POST /recommend

Public profiles use clean username URLs:

- /[username]
- /[username]/[endpointSlug]

Examples:

- /igal
- /igal/collaborate
- /demo
- /demo/collaborate

The core idea:

Links show where someone is.
OpenMe shows what people can do with them.

## MVP Features

1. User account
2. Public profile page
3. Profile links
4. Custom endpoints
5. Endpoint fields/questions
6. Endpoint boundaries
7. Visitor submissions
8. Owner inbox
9. AI analysis later:
   - summary
   - intent
   - boundary check
   - suggested reply
10. Public machine-readable profile API

## Non-MVP

Do not build yet:

- payments
- custom domains
- mobile app
- marketplace
- teams
- browser extension
- complex analytics
- advanced automations
- external calendar/email integrations
- Nuxt studio
